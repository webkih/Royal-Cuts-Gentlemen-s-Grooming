# Royal Cuts Gentlemen's Grooming Website

A clean, responsive one-page barbershop website for **Royal Cuts Gentlemen's Grooming** in Cottage City, Maryland.

## Business Information

- **Name:** Royal Cuts Gentlemen's Grooming
- **Rating:** 4.5 (39 reviews)
- **Address:** 3824 Bladensburg Rd, Cottage City, MD 20722, United States
- **Phone:** +1 (240) 714-5505
- **Plus Code:** W3P2+63 Cottage City, Maryland, USA

## What's Included

- Hero section with clear call-to-action buttons
- Services section for grooming offerings
- About section with brand-focused messaging
- Areas served and embedded map
- Photo gallery
- Customer testimonials slider
- Contact section and footer business details

## Project Structure

- `index.html` — Main page content and layout
- `styles.css` — Styling and responsive design
- `script.js` — Navigation, scroll, and testimonial interactions
- `img/` — Website image assets

## Run Locally

Because this is a static site, you can run it with a simple local web server:

```bash
python -m http.server 4173
```

Then open:

- `http://localhost:4173/index.html`

## Customize Content

To adapt this site for another business, update:

- Page title and meta description in `index.html`
- Business name and contact details across hero, contact, and footer sections
- Google Maps embed URL and directions links
- Testimonial copy and service descriptions

## Notes

- This project does not require a build step.
- You can deploy it to any static host (Netlify, Vercel static output, GitHub Pages, etc.).
